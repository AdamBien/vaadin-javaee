vaadin-javaee
=============

Vaadin Java EE Integration

Vaadin 6.8 can be seamlessly integrated with CDI and EJB by extending the AbstractApplicationServlet and injecting the com.vaadin.Application as @SessionScoped managed bean. See all (seven) steps required to create a Java EE 6 Maven 3 application for vaadin 6.8.X from
[scratch.](http://www.adam-bien.com/roller/abien/entry/vaadin_java_ee_no_xml)
